# 攀岩游戏设计文档（GDD）

> 本文档面向策划/设计师，描述游戏机制、操作方式和设计参数，不涉及代码实现细节。

---

## 一、游戏概述

这是一款**2D物理模拟攀岩游戏**。玩家操控一个攀岩者的四肢和躯干，在岩壁上抓取岩点、移动身体、保持平衡，从起点攀爬到终点。

**核心体验**：像真实攀岩一样，你需要思考手脚的放置顺序、身体重心分配、以及如何在岩壁上保持平衡。

---

## 二、操作方式

### 2.1 选中身体部位

| 操作 | 效果 |
|------|------|
| 点击 UI 按钮（左手/右手/左脚/右脚/躯干） | 选中该部位 |
| 键盘 Q / E / A / D / S | 分别选中：左手/右手/左脚/右脚/躯干 |

选中后，摇杆和方向键将控制该部位的移动。

### 2.2 移动

| 操作 | 效果 |
|------|------|
| 摇杆拖动 | 移动当前选中的肢体或躯干 |
| 摇杆推到边缘并保持 | 持续朝该方向移动（有加速度） |

### 2.3 抓握与释放

| 操作 | 效果 |
|------|------|
| 将手/脚拖动到岩点附近 | **自动吸附**（手/脚锁定到岩点上） |
| 拖动已吸附的手/脚超出释放范围 | **脱离岩点** |
| 双击部位按钮 / 键盘双击 Q/E/A/D | **强制释放**该部位的岩点 |

### 2.4 特殊操作

| 操作 | 效果 |
|------|------|
| 键盘 W | **回正身体**：逐步减小身体侧倾角度 |
| 键盘 R | **完全重置**：角色回到初始姿势 |
| 双击躯干按钮 / 键盘双击 S | **切换身体跟随模式**（手臂拉身体 vs 手臂独立移动） |

## 三、游戏流程

### 3.1 整体流程

```
初始站姿 → 起步（四肢锁定起步点）→ 攀爬过程 → 到达终点（双手锁定终点）→ 完成
```

### 3.2 起步条件

- 四肢（左手、右手、左脚、右脚）**全部**锁定在岩点上
- 每个岩点必须是**起步点**（`isStartPoint = true`）
- 手臂必须在岩点允许的**受力角度**内
- 脚部岩点必须支持**站立**或**钩挂**

### 3.3 终点条件

- 双手锁定在**同一个终点岩点**上
- 终点岩点标记为 `isFinishPoint = true`
- 双手均在受力角度内

## 四、岩点系统

### 4.1 岩点类型

| 类型 | 描述 | 手可用 | 脚可踩 | 脚可勾 |
|------|------|--------|--------|--------|
| **JUG（大把手）** | 舒适的大抓点 | ✅ | ✅ | ✅ |
| **POCKET（指洞）** | 手指扣入的孔洞 | ✅ | ✅ | ❌ |
| **CRIMP（ crimp）** | 指尖抠住的小边缘 | ✅ | ✅ | ❌ |
| **VOLUME（体积/长条）** | 长条形岩点，可滑动 | ✅ | ✅ | ✅ |

### 4.2 岩点属性

| 属性 | 说明 |
|------|------|
| **吸附半径** | 手脚靠近岩点多远时自动吸附 |
| **释放半径** | 手脚拖离岩点多远时脱离 |
| **冷却时间** | 脱离后多长时间内不能重新吸附该岩点 |
| **受力方向范围** | 手臂拉力的允许扇形角度（向下 ± 角度） |
| **起步点标记** | 是否可作为起步的岩点 |
| **终点标记** | 是否可作为终点的岩点 |

### 4.3 VOLUME（长条岩点）特殊机制

- 手在 VOLUME 上可以**沿线段滑动**
- 锁定后手的 Y 坐标可以自由调整，X 自动约束在 VOLUME 线段上
- 脱离判定基于离开 VOLUME 线段的距离

## 五、身体移动规则

### 5.1 支撑系统

角色必须**有支撑**才能移动身体（躯干）：

| 支撑来源 | 条件 |
|----------|------|
| 🦶 已吸附的脚（支持站立的岩点） | 脚锁定在可站立的岩点上 |
| 🦶 站在地面上的脚 | 脚 Y 坐标在接地容忍范围内 |
| 🤚 锁定且受力的手 | 手锁定在岩点上，且拉力方向在允许范围内 |

- 支撑形成一个**X轴区间** `[最左支撑点, 最右支撑点]`
- 无支撑时，身体和躯干**无法移动**
- 支撑区间越宽，角色越稳定

### 5.2 身体约束

移动身体（躯干）时，会受到以下约束：

1. **腿长限制**：吸附的脚不能拉得太远，髋部必须在腿长范围内
2. **臂长限制**：吸附的手不能拉得太远，肩膀必须在臂长范围内
3. **地面脚不离开地面**：无岩点锁定时，站在地面的脚不能悬空
4. **髋不低于双脚连线**：髋部必须在双脚连线上方一定距离
5. **髋不低于脚**：髋部高度必须大于最高脚的高度 + 最小偏移

### 5.3 手臂拉身体（跟随模式）

当选中手并拖到超出臂长范围时，如果**身体跟随模式**开启：
- 手臂先伸到最大长度
- 剩余的移动距离会**拖动整个身体**跟随手的方向

如果关闭此模式，手臂超出范围后只会停在最大长度。

## 六、平衡系统

### 6.1 质心与支撑

- 角色有一个**质心**（Center of Mass），由头、躯干、四肢各部位加权计算
- 质心的 X 坐标必须在**支撑区间**内
- 如果质心超出支撑区间，开始累计**失衡时间**

### 6.2 失衡掉落

| 参数 | 默认值 | 说明 |
|------|--------|------|
| 失衡容忍时间 | 0.5 秒 | 质心超出支撑区间多久后掉落 |
| 手臂角度容忍时间 | 0.3 秒 | 手臂拉力超出允许角度多久后松手 |

掉落时：角色**完全重置**到初始姿势，所有吸附清除。

### 6.3 手臂受力角度

每个岩点有允许的**受力方向扇形**：
- 基准方向向下，允许向上下各偏转一定角度
- 手臂拉力方向（肘→腕/手）必须在该扇形内
- 超出角度会开始松手倒计时

---

## 七、躯干侧倾

### 7.1 什么是躯干侧倾

身体可以向左右倾斜（肩膀相对髋部水平偏移），范围有限。

### 7.2 何时产生侧倾

- 当手或脚的目标**超出肢体长度**时，身体会自动侧倾来辅助到达
- 最大侧倾距离：可配置（默认约 40 像素）

### 7.3 回正机制

- 身体移动后，系统会**自动尝试减小侧倾角度**
- 玩家可以按 **W 键**主动逐步回正
- 回正的限度：另一只锁定手仍能到达其岩点

### 7.4 视觉表现

- 躯干侧倾时，肩膀跟随倾斜，头部也随之偏移
- 髋部位置不变

---

## 八、地面系统

- 所有场景都有一个**地平线**（地面 Y 坐标）
- 未吸附到岩点的脚**不会穿过地面**（不能低于地面）
- 脚靠近地面一定范围内视为"站在地面上"，可作为支撑
- 站在地面上的脚可以水平滑动，但 Y 坐标锁定在地面

---

## 九、吸附细节

### 9.1 自动吸附

手脚拖到岩点附近时自动吸附：
- **点吸附岩点**（JUG/POCKET/CRIMP）：在吸附半径内自动锁定到岩点中心
- **VOLUME 岩点**：在扩展矩形范围内自动吸附到线段上最近点

### 9.2 释放机制

| 方式 | 说明 |
|------|------|
| 拖拽脱离 | 将手脚拖离岩点超过释放半径 |
| 双击释放 | 双击对应部位按钮，立即释放 |
| 角度超时 | 手臂拉力角度超出允许范围超过容忍时间，自动松手 |
| 失衡掉落 | 角色失衡后全部释放 |

### 9.3 吸附保持期

锁定岩点后有 **0.5 秒保持期**：
- 此期间小幅拖动不会脱离
- 防止刚抓住岩点就被意外的微小拖动弹开

### 9.4 冷却机制

- 脱离岩点后，该岩点进入冷却（默认 2 秒）
- 冷却期间无法重新吸附
- **例外**：因角度超时自然松手的岩点，该肢体可无视冷却重新抓取

## 十、设计参数清单

| 参数 | 默认值 | 类别 | 调参建议 |
|------|--------|------|----------|
| **吸附半径** | 40px | 岩点 | 越大越容易抓，降低难度 |
| **释放半径** | 60px | 岩点 | 越大越难脱离，增加容错 |
| **冷却时间** | 2.0s | 岩点 | 越长越需要精确操作 |
| **受力角度范围** | ±45° | 岩点 | 越大越宽松 |
| **失衡容忍时间** | 0.5s | 平衡 | 越长越简单 |
| **手臂角度容忍时间** | 0.3s | 平衡 | 越长越宽松 |
| **髋部连线最小距离** | 10px | 身体约束 | 控制髋部必须高于脚连线 |
| **脚髋最小高度差** | 30px | 身体约束 | 脚不能太接近髋部 |
| **最大躯干侧倾** | 40px | 身体 | 范围越大越灵活 |
| **身体跟随模式** | 开 | 操作 | 手臂拉动身体 vs 独立移动 |
| **吸附保持期** | 0.5s | 吸附 | 防止误脱离 |
| **接地容忍** | 10px | 地面 | 多大范围内算"站地上" |
| **支撑宽度（脚）** | 30px | 支撑 | 单脚支撑区间的半宽 |
| **支撑宽度（手）** | 24px | 支撑 | 单手支撑区间的半宽 |

## 十一、难度调节建议

| 难度维度 | 调节方式 |
|----------|----------|
| **降低难度** | 增大吸附半径、增大受力角度范围、延长失衡容忍时间、缩短冷却时间 |
| **提高难度** | 减小吸附半径、缩小受力角度范围、缩短失衡容忍时间、增长冷却时间 |
| **增加策略性** | 增加 VOLUME 岩点（允许滑动）、设置更复杂的岩点布局 |
| **新手友好** | 默认开启身体跟随模式、更多起步点 |

---

## 十二、关卡设计要点

1. **起步点**必须四肢都可以够到，且布局合理
2. **终点**通常放在较高位置，需要双手同时够到
3. 岩点间距应考虑角色臂长和腿长，避免出现无法到达的"死点"
4. 支撑区间是核心平衡机制——确保在关键位置有足够的支撑宽度
5. VOLUME 岩点可以提供更多容错空间（允许滑动调整）
6. 岩点的受力方向设定会影响玩家可以采用的手部姿态
